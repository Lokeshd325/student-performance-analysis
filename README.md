# student-performance-analysis

## Objective
Analyze student academic performance using Python and generate insights.

## Tech Stack
Python, Pandas, Matplotlib, Pytest, GitHub Actions, Docker

## Features
- Data analysis
- Visualization
- Unit testing
- CI/CD pipeline
- Dockerized application

## How to Run
python src/analysis.py
python src/analysis.py plot
pytest

## CI/CD
GitHub Actions runs tests automatically on each push.
